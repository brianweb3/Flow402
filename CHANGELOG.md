# Changelog

All notable changes to Flow402 will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-XX

### Added

- Initial release of Flow402
- Decentralized compute marketplace functionality
- HTTP 402 payment protocol integration
- User authentication with JWT and OAuth
- Resource offer and rental management
- Payment processing with mock and Solana providers
- Rust cryptographic library with WebAssembly support
- Usage metering and settlement
- Node heartbeat API
- Cost calculator for rentals
- Provider earnings calculator
- Database schema with Prisma ORM
- Docker and Docker Compose configuration
- Comprehensive test suite
- API documentation

### Security

- JWT token-based authentication
- Password hashing with bcrypt
- Input validation with Zod
- httpOnly cookies for session management
- CSRF protection for auth endpoints

### Documentation

- README with setup instructions
- API endpoint documentation
- x402 payment protocol setup guide
- Rust library documentation
- Contributing guidelines
- Security policy
